# JavaScript Amazing Script

This script prints the message "JavaScript is amazing" to the console.

## Requirements

- This script is designed to be run on Ubuntu 20.04 LTS using Node.js version 14.x.
- The script is semistandard compliant (version 16.x.x) following the rules of Standard + semicolons on top, with a reference to the AirBnB style.
- All files should end with a new line.
- The first line of the script is exactly `#!/usr/bin/node`.
- The script file should be executable (`chmod +x your_script_name.js`).
- The length of the script will be tested using `wc`.

## Usage

1. Ensure you have Node.js version 14.x installed on your system.
2. Make the script executable using the following command:
    ```
    chmod +x your_script_name.js
    ```
    Replace `your_script_name.js` with the name of your JavaScript file.
3. Run the script:
    ```
    ./your_script_name.js
    ```

## File Structure

- `your_script_name.js`: The JavaScript file containing the script code.
- `README.md`: This file providing information about the script and its usage.
